These are the steps you have to follow to run this gulp demo
Steps:

1.Install Node.js from http://nodejs.org/download/

2.Install gulp globally
	Open node.js command prompt and execute this command i.e "npm install gulp" 

3.Clone this project at your local machine.

4.Install Gulp in this project directory
	Open node.js command prompt,enter into project directory and execute this command  i.e. "npm install gulp --save-dev" 

5.Run command to install all tasks written in package.json file i.e. "npm install"
		

6.Run gulp command and you will see the optimized directories for Js,Css and Images.
